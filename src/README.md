# サンプルプログラムの解析
## 使い方
### コンパイル
```
$ javac -encoding utf-8 *.java
```
### 実行
```
$ java AntsApplication
```
## AntsApplication.java
アプリケーションとして動くときのエントリポイントが入っている.
## AntsApplet.java
アプレットとして動くときのエントリポイントが入っている.
